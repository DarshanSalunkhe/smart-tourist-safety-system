## 🎨 Design Patterns Used

### 1. Lift + Scale Pattern
Used for premium button interactions
```css
transform: translateY(-3px) scale(1.02);
```

### 2. Gradient Overlay Pattern
Used for subtle hover effects
```css
background: linear-gradient(135deg, var(--primary-subtle), transparent);
```

### 3. Icon Animation Pattern
Used for engaging micro-interactions
```css
transform: scale(1.15);
```

### 4. Ring Animation Pattern
Used for attention-grabbing effects
```css
@keyframes sosRing { ... }
```

### 5. Pulse Dot Pattern
Used for status indicators
```css
@keyframes pulseDot { ... }
```
